get_osm_identifier(full_atlas_identifier)
Get the OSM identifier from the full Atlas identifier. A full Atlas
identifier contains a country code and way-sectioning component.
 
Example:
Atlas ID: 222222001003
OSM ID: 222222
Country code: 001
Way section ID: 003
